Rustをいじってみることにした。

### Rust本体のインストール

[Rustup](https://www.rustup.rs)を使う。

### ツール類

Racer（コード補完）とrustfmt（フォーマッタ）をcargoでインストールする。

```
cargo install racer
cargo install rustfmt
```

### Vim plugin

`rust-lang/rust.vim`と`racer-rust/vim-racer`を入れる。
